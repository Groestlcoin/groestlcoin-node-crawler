# Bitcoin Node Crawler [WIP]

1. Install the dependencies
    ```bash
    poetry install
    ```
1. Spin up the virtual environment
    ```bash
    poetry shell
    ```
1. Run the crawler
    ![image](https://user-images.githubusercontent.com/1169974/55775967-27f5d900-5a69-11e9-8733-fb31f5add62d.png)
1. Run tests
    ```bash
    pytest
    ```
